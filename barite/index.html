Barita Grado Petrolero

Suministro especializado para lodos de perforación con certificación API

Barita de alta calidad para la industria petrolera

En ROC'S Servicios Mineros, suministramos barita de grado petrolero que cumple con las especificaciones más exigentes del American Petroleum Institute (API) para su uso en lodos de perforación. Nuestro mineral garantiza las propiedades físicas y químicas necesarias para operaciones de perforación exitosas.

Contamos con yacimientos propios y alianzas estratégicas que nos permiten asegurar un suministro constante y confiable, respaldado por rigurosos controles de calidad en cada etapa del proceso, desde la extracción hasta la entrega final.

Especificaciones API Certificadas

Nuestra barita cumple y supera las especificaciones establecidas por el API Spec 13A para materiales de lodos de perforación:

Propiedades físicas garantizadas:

  • Gravedad específica: ≥4.20 g/cm³ (mínimo API)
  • Residuo en malla 200: ≤3.0% máximo
  • Residuo en malla 325: ≤9.0% máximo
  • Humedad: ≤1.0% máximo
  • Partículas >75 μm: ≤30% máximo
  • Solubilidad en HCl: ≤250 mg/kg máximo

Composición química típica:

  • BaSO₄: >92% mínimo
  • SiO₂: <6.0%
  • CaCO₃: <2.5%
  • Metales pesados: Dentro de límites API

Aplicaciones en Lodos de Perforación

La barita es el agente densificante más utilizado en la industria petrolera por sus propiedades únicas y confiabilidad operativa:

Funciones principales:

  • Control de presión: Incrementa la densidad del lodo para controlar presiones de formación
  • Estabilidad del pozo: Mantiene las paredes del pozo estables durante la perforación
  • Prevención de reventones: Contrarresta presiones de gas y fluidos de formación
  • Lubricación: Reduce fricción en la tubería de perforación
  • Suspensión de recortes: Facilita el transporte de detritos a superficie
  • Enfriamiento del bit: Disipa calor generado durante la perforación

Ventajas competitivas:

  • Químicamente inerte: No reacciona con otros componentes del lodo
  • No abrasiva: Minimiza desgaste de equipos de perforación
  • Fácil dispersión: Se mezcla uniformemente en el sistema
  • Estable térmicamente: Mantiene propiedades a altas temperaturas

Sistema de Control de Calidad

Implementamos un riguroso sistema de control de calidad que garantiza la consistencia y confiabilidad de nuestra barita:

Proceso de verificación:

  • Muestreo sistemático: Toma de muestras cada 500 toneladas producidas
  • Análisis granulométrico: Distribución de tamaño por tamizado húmedo
  • Gravedad específica: Medición con picnómetro calibrado
  • Análisis químico: XRF para composición elemental completa
  • Pruebas de campo: Ensayos de densidad en lodos reales
  • Certificación externa: Laboratorios acreditados ISO 17025

Garantía de calidad: Cada lote viene acompañado de certificado de análisis con trazabilidad completa desde el yacimiento hasta la entrega final.
